# OpenClaw Home Assistant Addon (Dev)

Development version of the OpenClaw Home Assistant Addon.

## Installation

Add this repository to your Home Assistant instance:

```yaml
https://github.com/chillkiller/openclaw-ha-addon/
```

## Features

- Full OpenClaw integration
- Development builds with latest features
- Bonjour/mDNS support
- Web UI with landing page
